Data Analytics Project 1

Project Overview

This project demonstrates the fundamentals of Exploratory Data Analysis (EDA) and Feature Engineering using Python. The goal is to clean, analyze, and prepare a dataset for machine learning by handling missing values, detecting outliers, and creating meaningful features.

Objectives

- Load and inspect the dataset
- Explore the data using descriptive statistics
- Identify and handle missing values
- Detect and remove outliers using the IQR method
- Perform feature engineering
- Prepare a clean dataset for machine learning models

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

Project Workflow

1. Import the dataset
2. Perform Exploratory Data Analysis (EDA)
3. Analyze missing values
4. Handle missing data
5. Detect outliers using the IQR method
6. Remove outliers
7. Create new features
8. Export the cleaned dataset

Results

- Dataset successfully explored and analyzed
- Missing values handled effectively
- Outliers detected and removed
- New features created for better analysis
- Clean dataset prepared for machine learning
